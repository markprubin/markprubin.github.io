Antarctic Icebergs: Tracking Giants on the Move

Mapping the journeys of massive icebergs across the Southern Ocean

View Dashboard →

The Challenge

The US National Ice Center tracks enormous icebergs — some larger than cities — as they break off Antarctica and drift through the Southern Ocean. They've been recording data for years, but it's scattered across 547 individual files on their website.

What I Built

I created an automated system to gather, organize, and visualize every tracked iceberg's journey from 2014 to 2025. The result is an interactive map showing the paths of dozens of named icebergs as they drift, spin, break apart, and eventually melt away.

What It Shows

Each colored line traces a single iceberg's movement over time. You can see how some icebergs get caught in ocean currents and circle the continent for years, while others drift north and disappear. Some cluster together near the Antarctic Peninsula, while others venture far out into open waters. Click any iceberg name to highlight just its path.

Why It Matters

These icebergs aren't just scenic — they're massive floating hazards to shipping routes and scientific research vessels. They also tell us about ice shelf stability and ocean circulation patterns. By visualizing their movements, scientists, maritime operators, and researchers can better understand how these giants behave.

The Technical Achievement

This project required building an entire data pipeline: automatically downloading 547 files from a government website, cleaning and standardizing inconsistent data formats, combining everything into one unified dataset, and creating geographic visualizations that make complex movement patterns instantly understandable.

Main Iceberg Data Scraper

script.py click to expand

Comprehensive web scraping and data collection pipeline. Loads iceberg file references, uses Selenium WebDriver to automate browser interactions, scrapes the US Ice Center website, extracts product IDs from download URLs, and automatically downloads all 547 CSV files.

import os, re, time, pandas as pd, requests
from bs4 import BeautifulSoup
from selenium import webdriver
from selenium.webdriver.chrome.options import Options
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

df_input = pd.read_csv("iceberg_files.csv")
df_input.columns = df_input.columns.str.strip().str.lower()
df_input["filename"] = df_input["filename"].str.strip().str.lower()

options = Options()
options.add_argument("--headless")
driver = webdriver.Chrome(options=options)
driver.get("https://usicecenter.gov/Products/ArchiveSearchMulti?...")

WebDriverWait(driver, 15).until(
    EC.presence_of_element_located((By.ID, "startdate-pick"))
)
start_date_input = driver.find_element(By.ID, "startdate-pick")
start_date_input.clear()
start_date_input.send_keys("11/07/2014")
driver.find_element(By.ID, "search").click()
WebDriverWait(driver, 15).until(
    EC.presence_of_element_located((By.ID, "tbl-berg"))
)

soup = BeautifulSoup(driver.page_source, "html.parser")
driver.quit()

records = []
for a in soup.find_all("a", href=True):
    text = a.get_text(strip=True)
    href = a["href"]
    if text.endswith(".csv") and "DownloadArchive?prd=" in href:
        match = re.search(r"prd=(\d+)", href)
        if match:
            records.append({"filename": text.strip().lower(), "prd": match.group(1)})

df_final = pd.merge(df_input, pd.DataFrame(records), on="filename", how="left")
df_final.to_csv("iceberg_files_with_prd.csv", index=False)

output_folder = "iceberg_csvs"
os.makedirs(output_folder, exist_ok=True)
for i, row in df_final.iterrows():
    if pd.notnull(row["prd"]):
        response = requests.get(f"https://usicecenter.gov/File/DownloadArchive?prd={row['prd']}")
        if response.status_code == 200:
            with open(os.path.join(output_folder, row["filename"]), "wb") as f:
                f.write(response.content)

Data Consolidation

merge_iceberg_csvs.py click to expand

Combines 547 individual CSV files into a single unified dataset. Scans directory for all CSVs, concatenates DataFrames with column alignment, handles missing data, and adds source file tracking for data lineage.

import pandas as pd
import os
from glob import glob

folder_path = "iceberg_csvs"
output_file = "antarctic_icebergs_combined.csv"

csv_files = glob(os.path.join(folder_path, "*.csv"))
print(f"Found {len(csv_files)} CSV files.")

dataframes = []
for file in csv_files:
    try:
        df = pd.read_csv(file)
        df["source_file"] = os.path.basename(file)
        dataframes.append(df)
    except Exception as e:
        print(f"Failed to read {file}: {e}")

df_combined = pd.concat(dataframes, axis=0, ignore_index=True, sort=True)
df_combined.to_csv(output_file, index=False)
print(f"Saved '{output_file}' with shape {df_combined.shape}")

Data Standardization

fix_csv.py click to expand

Reorders and standardizes column structure: Iceberg, Last Update, Latitude, Longitude, Length/Width, Area metrics, Remarks. Adds missing columns with null values and enforces consistent schema.

import pandas as pd

input_file = "antarctic_icebergs_combined.csv"
output_file = "antarctic_icebergs_combined_reordered.csv"

desired_order = [
    "Iceberg", "Last Update", "Latitude", "Longitude",
    "Length (NM)", "Width (NM)",
    "Area (sqKM)", "Area (sqMI)", "Area (sqNM)",
    "Remarks", "source_file"
]

df = pd.read_csv(input_file)
for col in desired_order:
    if col not in df.columns:
        df[col] = None

df = df[desired_order]
df.to_csv(output_file, index=False)
print(f"Reordered CSV saved to: {output_file}")

File Validation

search.py click to expand

Validates downloaded files against expected list. Compares expected vs. actual downloads, identifies missing files, generates completion reports, and ensures data pipeline integrity.

import os
import pandas as pd

download_folder = "iceberg_csvs"
expected_csv = "iceberg_files_with_prd.csv"
output_txt = "missing_files.txt"

df_expected = pd.read_csv(expected_csv)
df_expected["filename"] = df_expected["filename"].str.strip().str.lower()

downloaded_files = [
    f.lower() for f in os.listdir(download_folder)
    if f.endswith(".csv")
]

missing = df_expected[~df_expected["filename"].isin(downloaded_files)]

if not missing.empty:
    with open(output_txt, "w") as f:
        for name in missing["filename"]:
            f.write(name + "\n")
    print(f"Missing {len(missing)} file(s). See {output_txt}.")
else:
    print("All expected files are present.")
